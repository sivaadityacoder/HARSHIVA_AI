##  **Quick Start**

### **Prerequisites**
- Python 3.12+
- 8GB+ RAM (for AI model)
- Linux/macOS environment

### **Installation**
```bash
git clone <repository-url>
cd harshiva-ai
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### **Download AI Model**
```bash
mkdir models
cd models
# Download CodeLlama-7B-Instruct (4.08 GB)
```
